


netstat [options]


| فلگ  | توضیح                               |
| ---- | ----------------------------------- |
| `-t` | فقط TCP                             |
| `-u` | فقط UDP                             |
| `-l` | فقط لیسن‌شده‌ها (listening sockets) |
| `-n` | نمایش IP و پورت به‌جای نام          |
| `-p` | نمایش PID/نام برنامه                |
| `-a` | همه‌ی کانکشن‌ها (لیسن‌شده + فعال)   |
